

## Прайс-лист ветеринарной клиники ZooStatus

Создаём статическую страницу прайса из трёх файлов: `index.html`, `styles.css`, `app.js`.

### Данные и логика
- Загрузка двух CSV через PapaParse (CDN): `price.csv` (актуальные цены) и `newpricepopular.csv` (порядок отображения)
- Порядок услуг строго по "Ранг популярности" из `newpricepopular.csv`
- Цены всегда из `price.csv`, матчинг по ключу (Название + Подразделение + Категория), fallback по уникальному Названию, иначе "—"
- Услуги из `price.csv`, отсутствующие в файле порядка, добавляются в блок "Прочие услуги" (алфавитно)

### UI
- Шапка с заголовком "Прайс" в стиле ZooStatus (светлый, современный, аккуратный)
- Строка поиска с дебаунсом, фильтры по Подразделению и Категории
- Счётчик найденных позиций
- Карточка услуги: название, цена (формат ru-RU с разделителями), подразделение/категория мелким текстом
- Состояния: загрузка (спиннер), ошибка, пустой результат
- Адаптивная вёрстка (мобильные устройства)

### Техническая часть
- Настройки CSV URL и cache-bust вынесены в константы в начале `app.js`
- PapaParse через CDN для надёжного парсинга CSV (кавычки, запятые)
- Эффективный рендер через DocumentFragment для 2000+ строк
- Дебаунс поиска (300мс)

### Файлы
- `public/index.html` — HTML-страница с подключением PapaParse CDN, styles.css и app.js
- `public/styles.css` — стили в стиле ZooStatus
- `public/app.js` — вся логика загрузки, парсинга, матчинга и рендера
- Копирование CSV-файлов в `public/price.csv` и `public/newpricepopular.csv`
