# ZooStatus — Прайс-лист

Веб-приложение для отображения актуального прайс-листа ветеринарной клиники ZooStatus.

## Технологии

- Vite
- TypeScript
- React
- Tailwind CSS
- PapaParse (CSV-парсинг)

## Запуск

```sh
npm i
npm run dev
```

## Разработка и релизы (как работать с репозиторием)

- **`dev`**: ветка для частых изменений (в том числе из Lovable).
- **`main`**: только стабильные изменения, которые можно выкатывать на прод.

Рекомендуемый процесс:

- **Разработка**: пушите изменения в `dev`.
- **Стабилизация**: делайте Pull Request `dev` → `main` (CI проверит `lint/test/build`).
- **Выкатка**: сисадмин деплоит либо `main`, либо **теги релизов** `vX.Y.Z` (рекомендуется).
